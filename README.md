# Kinopoisk Bot
This project allows users to search for links to movies and send them via an inline Telegram bot. Rewritten in Rust.


## Usage:
To use the bot, simply type @kp_bobot in any chat and start typing the name of the movie you are looking for. The bot will show a list of movie links that match your query.

## Installation
1. Edit `.env` file.
2. `docker build .`
3. `docker compose up`
